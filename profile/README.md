## Trust Verification Layer for AI Agents

**Veritera** builds [Forge](https://veritera.ai) — the infrastructure that verifies every AI agent action before it executes. Define policies, check actions in real time, get cryptographic proof of every decision.

### SDKs & Integrations

| Package | Install | Framework |
|---------|---------|-----------|
| [forge-python](https://github.com/VeriteraAI/forge-python) | `pip install veritera` | Any Python agent |
| [forge-js](https://github.com/VeriteraAI/forge-js) | `npm i @veritera.ai/forge-verify` | Any JS/TS agent |
| [forge-openai](https://github.com/VeriteraAI/forge-openai) | `pip install forge-openai` | OpenAI Agents SDK |
| [forge-langchain](https://github.com/VeriteraAI/forge-langchain) | `pip install langchain-forge` | LangGraph / LangChain |
| [forge-crewai](https://github.com/VeriteraAI/forge-crewai) | `pip install crewai-forge` | CrewAI |
| [forge-llamaindex](https://github.com/VeriteraAI/forge-llamaindex) | `pip install llama-index-tools-forge` | LlamaIndex |

### How It Works

```
Your AI Agent → Forge Verify → Policy Check → Approved / Denied → Cryptographic Proof
```

Every decision is logged. Every action is verified. Every proof is auditable.
